🛠️ Installation

1. Clone the repository:

git clone https://github.com/abhishekakhand737/Python.git

2. Navigate to folder:

cd Python

3. Install required libraries:

pip install numpy pandas matplotlib seaborn

---

▶️ How to Run

Run any Python file:

python filename.py

Or use Jupyter Notebook:

jupyter notebook
